# Multiple Choice Questions

Multiple Choice Questions is a web and desktop study application designed for practicing exam-style multiple-choice questions.

It helps you upload your own question sets, solve questions by topic, track your progress, and continue studying from where you left off.

## Features

- Upload question sets from JSON, Markdown, text, and supported APKG files
- Import supported question files directly from Google Drive in the web version
- Select one or more question sets and study them together
- Filter questions by subject or topic
- Shuffle questions to avoid memorizing the order
- Mark answers and review correct, incorrect, or unanswered questions
- Continue from where you left off with automatic progress saving
- Reopen the same question set later and keep your previous progress
- Review only the questions you answered incorrectly
- Lock answers after selection to prevent accidental changes
- Automatically move to the next question after answering
- Edit supported question sets inside the app
- View study summaries, activity trends, and topic-based performance insights
- Use keyboard shortcuts for faster studying
- Switch between light and dark mode
- Use full-screen mode for a more focused study session
- Print tests or save them as clean A4 PDF files
- Check for updates in the desktop version

## Usage

1. Open the app.
2. Upload a question set.
3. Select the set or sets you want to study.
4. Start solving questions.
5. Choose an answer, review the explanation, and continue.

Keyboard shortcuts:

- `A` to `E`: Select an answer choice
- `S`: Show or hide the explanation
- `←` and `→`: Move between questions
- `F`: Enter full-screen mode
- `Esc`: Exit full-screen mode
